# OOAD_SmartCampus_Project

# Smart Campus Facility Booking & Maintenance System

## Project Overview
This project is a comprehensive solution designed to automate and streamline the management of campus facilities at Universities. 
The system enables event organizers to book venues, allows facility managers to oversee allocations, and provides administrators with tools for system maintenance and reporting.

Built using **Object-Oriented Analysis and Design (OOAD)** principles, the application adheres to the **MVC Architecture** and incorporates various **Design Patterns** to ensure scalability and maintainability.

---

## Key Features
The system supports three primary user roles, each with specific modules:

### 1. Event Organizer (Club Head)
* **Manage Event Booking:** Request venue slots and upload event details.
* **Booking Control:** Modify or cancel existing requests.
* **Maintenance:** Inform or assign maintenance for venues.

### 2. Facility Manager
* **Allocation Control:** Approve or reject incoming venue requests.
* **Rule Management:** Set and enforce specific venue rules.
* **Monitoring:** Monitor facility usage and status.

### 3. Administrator (System Manager)
* **User & Facility Management:** CRUD operations for system users and campus resources.
* **Master Schedule:** Maintain a centralized schedule to prevent booking conflicts.
* **System Health:** Generate reports and manage database backups.
